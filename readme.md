## 🌟 Features

### 🎬 Cinematic Intro Experience
- **Animated boarding pass** with shimmer effects and live date
- **3D floating plane** icon with smooth flight animations
- **Gold gradient text** with shifting color animations
- **Staggered fade-in** entrance sequence
- **Glassmorphism design** with backdrop blur

### 📚 Complete Career Guide
| Section | Content |
|---------|---------|
| **Requirements** | Citizenship, age, height, BMI, education, eyesight, tattoos, languages |
| **Application Process** | 6-phase timeline from online application to graduation |
| **Grooming Standards** | Full female & male grooming rules with detailed breakdowns |
| **Salary & Benefits** | Detailed salary table for Malaysia Airlines, AirAsia, Batik Air, Firefly |

### 🏢 Malaysian Airlines Database
- **Malaysia Airlines** — Full-Service National Carrier
- **AirAsia** — World's Best Low-Cost Carrier
- **Batik Air Malaysia** — Full-Service Regional
- **Firefly** — Community Airline

### 🎓 Training Modules (6 Core)
1. **Safety & Emergency Procedures (SEP)** — 2 weeks, 100% pass required
2. **First Aid & CPR (AED)** — Recertify every 12 months
3. **In-Flight Service & F&B** — Practical assessment included
4. **Aircraft Familiarization** — Type-specific per aircraft
5. **Communication & Customer Service** — Continuous assessment
6. **Ditching & Water Survival** — Must pass swimming test

### 🎮 3D Flight Simulator
- **Three.js powered** 3D airplane with engine glow and cloud particles
- **10 realistic scenarios** covering emergencies, service, and safety
- **Scoring system** with streak bonuses
- **Gold / Silver / Bronze** badges based on performance
- **Detailed feedback** for every answer

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN libraries)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/skycrew-academy.git

# Navigate to project directory
cd skycrew-academy

# Open in browser
open index.html
```

Or simply double-click `index.html` to launch!

## 🎨 Design System

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Navy | `#0a1628` | Background |
